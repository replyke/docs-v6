---
title: "Fetch Connections Count"
api: "GET /:projectId/api/v7/connections/count"
description: "Get the number of accepted connections for the authenticated user"
---

Returns the total number of accepted connections for the authenticated user.

## Response

On success, returns HTTP `200`:

<ResponseField name="count" type="number">
  Total number of accepted connections for the authenticated user.
</ResponseField>
