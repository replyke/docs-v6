---
title: "Fetch Followers Count"
api: "GET /:projectId/api/v7/follows/followers-count"
description: "Get the number of followers the authenticated user has"
---

Returns the total number of users following the authenticated user.

## Response

On success, returns HTTP `200`:

<ResponseField name="count" type="number">
  Total number of users following the authenticated user.
</ResponseField>
