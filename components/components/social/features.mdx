# Social Comments Features

Instagram-style comment system features.

---

## Core Features

### Single-Level Nesting

Maximum 2 levels: comments and replies.

```
Comment 1
  └─ Reply to Comment 1
  └─ Reply to Comment 1

Comment 2
  └─ Reply to Comment 2
```

Keeps conversations simple and easy to follow.

---

### Heart/Like System

Positive-only engagement with heart icons.

**Features:**
- Heart button (❤️)
- Like count display
- Only one like per user per comment
- Unlike capability
- No downvotes

---

### Sorting

Built-in Top/New/Old sorting.

**Options:**
- **Top:** Most liked comments first
- **New:** Newest comments first
- **Old:** Oldest comments first

Users can switch sorting on the fly.

---

### @ Mentions

Tag users with autocomplete, same as threaded comments.

---

### Edit & Delete

Authors can edit/delete their comments.

---

### Reporting

Report inappropriate comments via action menu.

---

### Dark Mode

Full dark mode support (inline styles or Tailwind).

---

### Real-Time Updates

Automatic updates when new comments arrive.
